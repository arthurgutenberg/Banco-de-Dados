Estrutura de um documento html

A estruturra de um documento html e definida pela declaracao !DOCTYPE html, pelas tags html, head e body.

Essa estrutura e interpretada pelo navegador que analisa cada tag e renderiza para o usuario.

A tag head e utilizada para configuracoes do navegador e consequentemente do documento HTML, as informacoes contidas dentro dessa tag sao invisiveis para o usuario, com excecao do titulo que ficara visivel na parte superior da pagina.

Ja a tag body e utilizada para exibir toda parte visivel para o usuario e nela teremos outras tags que tambem serao utilizadas pelo navegador.

Tag HTML, essa tag pe a raiz que cotem todas as outras tags do documento html, delimitando a totalidade do documento.

Por fim ainstrucao DOCTYPE html, que tem a finalidade de indicar ao navegador qual a versao do html que sera utilizado. Ela deve vir na primeira linha do documento.
